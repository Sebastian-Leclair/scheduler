Title: Customer Appointment Scheduler

Purpose: The purpose of this application is to allow users to manage appointments and customers from a database.

Author: Kol Leclair

Contact Information: klecla5@wgu.edu

Version: 1.0

Date: 05/19/2023

IDE: IntelliJ Community 2022.3.3

JDK: Java 17.0.6

JavaFX: javafx-sdk-20.0.1

Directions:
1. Set up your MySQL database with the settings that are found in the JDBC class.
2. Start the application.
3. Login with valid credentials.
4. If the login is successful, you will be able to manage appointments and customers from the database as needed.

Additional Report:
The additional report shows all the customers that are part of the selected country.

MySQL Connector Driver: mysql-connector-j-8.0.33
